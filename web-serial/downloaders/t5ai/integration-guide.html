3. 代码示例

基本使用

// 创建T5AI下载器
const downloader = new T5Downloader(serialPort, (level, message) => {
  console.log(`[${level}] ${message}`);
});

// 连接设备（自动处理跨平台复位）
await downloader.connect();

// 下载固件
await downloader.downloadFirmware(firmwareData, 0x00000000);

高级配置

// 手动控制Transport层
const transport = new T5AITransport(serialPort, debugCallback);
await transport.connect();

// 执行跨平台复位
await transport.reset();

// 获取平台信息
const platformInfo = transport.getPlatformInfo();
console.log('平台:', platformInfo.platform);

5. 故障排除

提示: 如果在Ubuntu上遇到问题，请确保：
  • 用户在dialout组中: sudo usermod -a -G dialout $USER
  • 注销并重新登录生效
  • 使用Chrome/Edge浏览器（Firefox不支持Web Serial API）
